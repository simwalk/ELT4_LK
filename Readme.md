**Fehler:**
Es sind sicher noch Fehler in den Lernkarten vorhanden. Diese dürfen gerne gemeldet werden, damit diese Korrigiert werden können. Allenfalls kann man diese auch mit einem Pull-Request selbst korrigieren.
